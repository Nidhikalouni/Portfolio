## 📦 Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Nidhikalouni/Portfolio.git
   cd Portfolio
2. **Install Dependencies**
   npm install
3. **Start Development Server**
   npm run dev
4. **Build for Production**
   npm run build
